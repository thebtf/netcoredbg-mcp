## ADR Template

Create new ADRs using this template:

```markdown
# ADR-NNN: Title

**Status:** Proposed | Accepted | Deprecated | Superseded
**Date:** YYYY-MM-DD
**Supersedes:** ADR-XXX (if applicable)

## Context

What is the issue that we're seeing that is motivating this decision?

## Decision

What is the change that we're proposing and/or doing?

## Options Considered

### Option 1: [Name]
- Pros: ...
- Cons: ...

### Option 2: [Name]
- Pros: ...
- Cons: ...

## Consequences

### Positive
- ...

### Negative
- ...

## Implementation Notes

Key implementation details or considerations.
```

## Guidelines

1. **One decision per ADR** — Keep focused
2. **Record context** — Why was this needed?
3. **List alternatives** — What else was considered?
4. **Note consequences** — Both positive and negative
5. **Update status** — Keep ADRs current
